# 🖐️ Pasos para Contribuir

## 1️⃣ Elegir un Issue

Selecciona un issue disponible en GitHub:

```bash
Issues
```

Luego comenta que deseas trabajar en él para evitar conflictos con otros colaboradores.

---

## 2️⃣ Crear una Rama

Crea una nueva rama siguiendo las convenciones del proyecto:

```bash
git checkout -b feature/nueva-funcion
```

Ejemplos:

```bash
feature/player-ui
fix/login-error
docs/readme-update
```

---

## 3️⃣ Realizar Cambios

Haz las modificaciones necesarias:

- ✨ Nuevas funciones
- 🐛 Fixes
- 🎨 UI improvements
- 📚 Documentation

---

## 4️⃣ Agregar Tests

Toda nueva funcionalidad debe incluir pruebas cuando sea posible.

```bash
npm test
```

---

## 5️⃣ Commit de Cambios

Usa commits claros y organizados:

```bash
git commit -m "✨ Add new playlist feature"
```

Ejemplos:

```bash
🐛 Fix playback issue
🎨 Improve sidebar UI
📚 Update documentation
⚡ Optimize API requests
```

---

## 6️⃣ Push al Repositorio

```bash
git push origin feature/nueva-funcion
```

---

## 7️⃣ Crear Pull Request

Abre un Pull Request en GitHub y completa la plantilla correspondiente.

Incluye:

- 📌 Descripción clara
- 📸 Screenshots si aplica
- 🧪 Tests realizados
- ⚡ Cambios importantes
