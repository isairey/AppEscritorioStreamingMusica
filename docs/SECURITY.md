# 🚨 Reportar Vulnerabilidades

Si encuentras una vulnerabilidad o problema de seguridad, por favor repórtalo de manera responsable.

## 📩 Información que debes incluir

Al enviar un reporte intenta proporcionar:

- 🐛 Descripción detallada del problema
- 🔁 Pasos para reproducir la vulnerabilidad
- ⚠️ Impacto potencial
- 💡 Posibles soluciones o mitigaciones
- 📸 Evidencia o screenshots (si aplica)

## 🛠️ Resolución

Si la vulnerabilidad es confirmada:

- 🔥 Se desarrollará una corrección
- 🚀 Se liberará un parche de seguridad
- 📦 Se publicará una actualización

Tiempo estimado:

```bash
14 - 30 días
```

Dependiendo de la complejidad del problema.
